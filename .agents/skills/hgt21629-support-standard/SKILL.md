# HG/T21629 管架标准图 Skill

HG/T21629-2021《管架标准图》是化工、石油化工等工业装置中工艺管道、公用工程管道和烟气管道标准管架选用的重要标准。

---

## 标准概述

| 项目 | 内容 |
|------|------|
| 标准号 | HG/T 21629-2021 |
| 中文名称 | 管架标准图（附条文说明） |
| 英文名称 | Piping support standard drawings |
| 发布日期 | 2021-05-17 |
| 实施日期 | 2021-10-01 |
| 替代标准 | HG/T 21629-1999 |
| 页数 | 353页 / 402页 |
| 适用范围 | 化工、石油化工等工业装置的工艺管道、公用工程管道和烟气管道 |

## 材料要求

### 常用材料

| 材料牌号 | 类型 | 适用温度 | 应用场景 |
|----------|------|----------|----------|
| Q235B | 碳素结构钢 | -20℃~300℃ | 一般管架 |
| Q355B | 低合金钢 | -40℃~400℃ | 重要管架 |
| 20# | 优质碳素钢 | -20℃~425℃ | 管道、螺栓 |
| 16Mn | 低合金钢 | -40℃~450℃ | 低温工况 |
| 15CrMo | 合金钢 | ≤550℃ | 高温工况 |
| 0Cr18Ni9 | 不锈钢 | -196℃~700℃ | 腐蚀/高温 |
| PTFE | 聚四氟乙烯 | -200℃~260℃ | 滑动垫片 |

### 材料选用原则

1. **与管道接触部件**：按管道设计温度选用
2. **不接触管道部件**：按环境温度选用
3. **高温工况(>425℃)**：选用耐热合金钢
4. **低温工况(<-20℃)**：选用低温钢或不锈钢
5. **腐蚀环境**：选用不锈钢或防腐涂层

## 管架选用流程

```
1. 确定管道参数
   ├── 管径 DN
   ├── 设计温度
   ├── 设计压力
   ├── 管道材质
   └── 保温/保冷厚度

2. 确定支撑要求
   ├── 支撑位置
   ├── 支撑方式(吊/托/支)
   ├── 位移要求
   └── 荷载大小

3. 选择管架类型
   ├── 生根方式(梁/柱/地面/设备)
   ├── 结构形式
   └── 特殊要求(减振/导向/限位)

4. 确定管架编号
   ├── 按标准图选用
   └── 校核荷载和尺寸

5. 材料统计
   ├── 零件拆分
   ├── 材料汇总
   └── 防腐面积计算
```

## 注意事项

1. **管子外径系列**：本标准按GB/T28708的管子外径系列Ⅰ编制，用于系列Ⅱ需调整
2. **温度范围**：选用时注意管架材料的温度适用范围
3. **荷载校核**：必须校核管架的允许荷载是否满足要求
4. **防腐要求**：根据环境条件确定防腐等级和措施
5. **制作安装**：应符合标准的制作和安装要求
